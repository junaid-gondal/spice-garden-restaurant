## Recommended Dimensions
- Hero: 1920x1080 or larger
- Menu items: 800x800 (square, food photography)
- Gallery: 800x600 or larger
- People: 400x400 (square portraits)
- Logo: 512x512 PNG with transparency

## After Adding Images
Update image paths in:
- src/data/menuData.js
- src/data/featuredDishes.js
- src/data/testimonials.js
- src/data/gallery.js
- All Hero components
